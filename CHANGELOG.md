## 0.1.0

- Updated dependencies.
- Fixed a bug where boolean attributes sometimes did not display their
  diagnostics correctly.
- Added a new setting `keml.warnOnLogAttribute` that controls the display of
  warnings generated by the use of the new `log` attribute.
- Added support for the following new attributes from KEML v3.3:
  - `sse` – real-time data pushed from the server to the client
  - `scroll` – automatic scrolling support
  - `relative` – affects `scroll`, making the scrolling relative rather than
    absolute
  - `behavior` – scrolling transition (animation)
  - `top` – the vertical scroll offset
  - `left` – the horizontal scroll offset
  - `log` – logging support

## 0.0.4

- Updated internal dependencies, including the core functional one of

## 0.0.3

- Cosmetic improvements to some diagnostic ranges.
- Implemented a robust, deterministic method for identifying
  **definitely valid** and **definitely invalid** action names.
- Added automatic cleanup for virtual documents when they are closed.

## 0.0.2

- fixed an attribute resolution bug
- removed edit batching, as it was more trouble than it is worth, because too

  for batching to be feasible
